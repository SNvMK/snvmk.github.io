What it is?

SNVMDB is a simplest and easiest database management tool.
It is written in Python and use JSON as databases format

How can I install it?

You can install it only from GitHub, because I`m too lazy to host ir to PyPi
So do the: pip install git+"https://github.com/SNVMK/snvmdb" And you need install Git and add it to path before install!

How can I use it?

After you install this, you can write a simple code!
Example: import snvmdb

db = snvmdb.Database("db.json")

students = db.create_table("students")
students.put("John", 21)
students.put("Suzie", 19)

content = db.show()
table_content = students.show()
print(content)
print(table_content)
What`s doing this code? Lets show line-by-line!

import snvmdb It is importing library that we installed

db = snvmdb.Database("db.json")
Initialize empty database and store it to variable.
You may need re-start code after this.

students = db.create_table("students") Create new table "students" and store it to variable.

students.put("John", 21)
students.put("Suzie", 19)
Put two pairs to table: John: 21 and Suzi: 19

content = db.show()
table_content = students.show()
print(content)
print(table_content)
Show all tables in database and items in table
